Agricultural Monitoring Board

Design

This Microcontroller Board was developed to help the farmers to get a deep understanding of the environmental factors affecting the growth of the crops. The device allows the user to track temperature, humidity, moisture content of the soil and light around the plant. System can alert the user if any condition falls out of optimal range, so the user can maximize the harvest yields.
